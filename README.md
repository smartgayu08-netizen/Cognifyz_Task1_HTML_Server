# Task 1 - HTML Structure and Basic Server Interaction

## Objective

To understand server-side rendering and basic form submission using Node.js, Express.js, and EJS.

## Technologies Used

* HTML
* Node.js
* Express.js
* EJS

## Features

* Student Registration Form
* Form Submission Handling
* Server-Side Rendering
* Dynamic Display of Submitted Data

## How to Run

1. Install dependencies:
   npm install

2. Start the server:
   node app.js

3. Open in browser:
   http://localhost:3000

## Output

The application allows users to enter their name and email through a form and displays the submitted information on a result page.
